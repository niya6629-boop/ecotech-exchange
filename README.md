# ♻️ EcoTech Exchange – Sustainable Student Marketplace

A high-fidelity marketplace prototype designed to promote sustainable technology reuse among verified students.

EcoTech Exchange aims to reduce electronic waste (e-waste) by providing a secure and trusted environment where students can buy, sell, and exchange electronic devices within an academic community. The project was developed as part of an interdisciplinary initiative combining Web Development and English communication skills.
